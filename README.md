## Supportive formats:

- json
- yml/yaml

## Output formats:

- stylish
- plain
- json

## Get more information about the program

gendiff --help

## Run program

- gendiff [filepath1] [filepath2] (output format is stylish by default)
- gendiff --format [format] [filepath] [filepath]
